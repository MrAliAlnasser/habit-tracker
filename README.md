## تحديثات (01-07-2026)

*وقت الإنجاز: 23:04*
### ⚙️ إعدادات العادات المتقدمة
- **العادات المؤقتة:** تمت إضافة خيار جديد في إعدادات العادة باسم "لهذا اليوم فقط". يتيح هذا الخيار إضافة مهام أو عادات تُنجز لمرة واحدة وتظهر في يوم إضافتها فقط، دون أن تتكرر في الأيام السابقة أو اللاحقة.

*وقت الإنجاز: 22:50*
### 🌟 تخصيص العادات وتحسين الواجهة
- **نوافذ منبثقة (Modals) أنيقة:** 
  - تم تحويل قائمة "التصنيفات" (عبادات، صحّة، إنتاجيّة) وقائمة "أيقونة العادة" إلى أزرار تفتح نوافذ منبثقة أنيقة في المنتصف بتأثير زجاجي (Blur)، مما استبدل القوائم المنسدلة التقليدية وجعل واجهة الإضافة أكثر فخامة.
  - تم استبدال إيموجي الموسيقى بإيموجي آخر مفيد (🥦).
- **تنسيق الأزرار:** تم نقل زر "إضافة" ليكون بجوار خيار "⚙️ إعدادات" في الأسفل، مما يجعل واجهة إضافة العادة أكثر ترتيباً وانسجاماً.

### ✨ إضافات
- تخصيص أيام العادة: اختيار تكرار العادة (يوميًا أو أيام محددة).

- شاشة تحميل (Splash Screen) وأيقونة مخصصة (Favicon).

### 🎨 تحسينات بصرية ومحتوى
- أصوات إنجاز: نغمة تحفيزية عند إكمال العادة.
- حركات تفاعلية: أنيميشن دخول سلس للعناصر والقوائم.
- واجهة أبسط: إزالة الصناديق الزائدة وتمييز زر "إضافة".
- تقويم أوضح: تعديل خطوط أسماء الأيام لمنع التداخل.

### ⚙️ تحسينات تقنية
- استجابة فورية: ظهور العادة الجديدة فوراً بدون تحديث الصفحة.
- استقرار الخيارات: إصلاح مشكلة اختفاء خيارات (تحديد الأيام).
- أداء سلس: إصلاح وميض (Flash) الأزرار عند الضغط.
